========================================
🎟 蓝牙嗅探器 (BLE Sniffer)
========================================

Nordic蓝牙嗅探器是基于Nordic半导体芯片开发的BLE协议分析工具，用于捕获和分析蓝牙低功耗设备的通信数据。

主要特性：
- 支持BLE 4.0/4.1/4.2/5.0协议
- 实时数据包捕获
- 协议解析与显示
- 与Wireshark集成分析
